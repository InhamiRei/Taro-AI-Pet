# Taro AI Pet

一个可爱的 AI 桌宠应用，会定时截图你的桌面并通过大语言模型（如百炼或自定义 AI）进行分析，像一个好奇的伙伴一样评论你正在做的事情。项目目前已集成 Live2D 模型展示与交互功能。

## 功能特性

- **Live2D 桌宠模型**，支持多种动画状态、表情与动作的交互切换。
- **定时截图分析**，定时获取桌面状态并发送给 AI 视觉模型。
- **对话气泡**，不仅有打字机效果显示 AI 回复，还会有傲娇等设定性格的回应。
- **透明可拖拽**，支持点击穿透，不影响你的正常办公体验。
- **丰富互动**，包括摸头互动、随机循环动画触发机制、特效展示等。
- **右键控制系统**，方便控制模型状态、切换表情与强制刷新。

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置环境变量
复制 `.env.example` 为 `.env`，填入你的 API 配置（如百炼 API Key）：
```
DASHSCOPE_API_KEY=sk-your-key-here
SCREENSHOT_INTERVAL=300000   # 截图间隔（毫秒），默认 5 分钟
AI_MODEL=qwen-vl-max         # 视觉模型
```

### 3. 启动应用
```bash
npm start
```

开发模式（带 DevTools）：
```bash
npm run dev
```

## 使用方法

| 操作 | 效果 |
|------|------|
| **拖拽模型** | 移动桌宠位置 |
| **单击模型** | 触发反应与特性，播放随机动作并中断休眠 |
| **双击模型** | 强制触发手动截图与 AI 观察分析 |
| **右键模型** | 呼出互动菜单，用于切换表情和播放动作 |

## 项目结构

```
src/
├── main/              # Electron 主进程
│   ├── index.js       # 应用入口
│   ├── screenshot.js  # 截图模块
│   ├── ai-service.js  # AI API 调用
│   ├── tray.js        # 系统托盘
│   └── config.js      # 配置管理
├── preload/
│   └── preload.js     # 安全的 IPC 桥接
└── renderer/          # 渲染进程（桌宠 UI）
    ├── index.html
    ├── css/style.css  # 样式和动画
    ├── js/app.js      # Live2D 交互核心逻辑
    └── models/        # Live2D 角色模型资产目录
```

## 注意事项

- macOS 首次运行需要授予「屏幕录制」权限（系统偏好设置 > 隐私与安全性）。
- 截图功能会将屏幕信息发送到配置的 API 服务，请注意个人隐私安全。
